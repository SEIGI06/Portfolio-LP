## Commandes rapides

### Système
```bash
# Vérifier l'espace disque
df -h

# Vérifier la mémoire
free -h

# Voir les processus en cours
ps aux
```

### Réseau
```bash
# Vérifier la configuration réseau
ip a

# Tester la connectivité
ping google.com

# Voir les ports ouverts
netstat -tuln
```
